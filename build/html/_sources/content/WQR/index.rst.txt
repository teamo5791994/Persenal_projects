===================================
Water Quality Retrieval Project
===================================

.. figure:: ./imgs/changjiang.png
   :width: 650px
   :align: center

This project is a case study of trying to develop a model that can retrieve water-quality level from Landsat-8 images.
The main achievement is the successful display the spatial water-quality level distribution in a large area.
